Correct Manual Handling Techniques

Hazard ID 11203
Control Icon
Description

Manual handling injuries can happen in any work envirnment. Heavy manual labour, awkward postures, manual materials handling, and previous or existing injury are all risk factors in developing musculoskeletal disorders (MSDs). Correct techniques must be used where the task is unavoidable and other methods are not practicable.
